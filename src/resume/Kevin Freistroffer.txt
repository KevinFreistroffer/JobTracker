1. Comment Length Calculator
Problem: Count non-space characters in source code, ignoring // line comments and /* */ block comments.

Solution
def solution(source):
    NORMAL, LINE, BLOCK = 0, 1, 2
    state = NORMAL
    count = 0
    text = '\n'.join(source)
    i = 0
    n = len(text)
    while i < n:
        c = text[i]
        if state == NORMAL:
            if c.isspace():
                pass
            elif c == '/' and i + 1 < n:
                if text[i + 1] == '/':
                    state = LINE
                    i += 2
                    continue
                elif text[i + 1] == '*':
                    state = BLOCK
                    i += 2
                    continue
                else:
                    count += 1
            else:
                count += 1
        elif state == LINE:
            if c == '\n':
                state = NORMAL
        elif state == BLOCK:
            if c == '*' and i + 1 < n and text[i + 1] == '/':
                state = NORMAL
                i += 2
                continue
        i += 1
    return count
How it was solved
Use a state machine with three modes:

State	Behavior
NORMAL
Count non-space chars; // → LINE, /* → BLOCK
LINE
Ignore until newline
BLOCK
Ignore until */
Join all lines with \n so block comments can span multiple lines, then scan left to right once.

Example: "int b = 47;/*37;*///41;"
→ count int b = 47; (8 chars), skip /*37;*/, skip //41; → 8

Total for the full example: 7 + 8 + 7 + 12 = 34

2. Hostname Tracker (allocate / deallocate)
Problem: Hostnames are {hostType}{number} (e.g. apibox1). allocate returns the lowest available number; deallocate releases it. Only implement the Tracker class — don't modify CodeSignal's existing solution() driver.

Solution
class Tracker:
    def __init__(self):
        self.next_num = {}
        self.free = {}
        self.allocated = {}
    def _parse(self, hostname):
        i = len(hostname) - 1
        while i >= 0 and hostname[i].isdigit():
            i -= 1
        return hostname[:i + 1], int(hostname[i + 1:])
    def _ensure(self, hostType):
        if hostType not in self.next_num:
            self.next_num[hostType] = 1
            self.free[hostType] = []
            self.allocated[hostType] = set()
    def allocate(self, hostType):
        self._ensure(hostType)
        if self.free[hostType]:
            num = min(self.free[hostType])
            self.free[hostType].remove(num)
        else:
            num = self.next_num[hostType]
            self.next_num[hostType] += 1
        self.allocated[hostType].add(num)
        return hostType + str(num)
    def deallocate(self, hostname):
        hostType, num = self._parse(hostname)
        if hostType not in self.allocated:
            return
        if num not in self.allocated[hostType]:
            return
        self.allocated[hostType].remove(num)
        self.free[hostType].append(num)
How it was solved
Per hostType, track three things:

next_num — next never-used number (starts at 1)
free — list of released numbers (reuse lowest first via min())
allocated — numbers currently in use (so invalid deallocates are no-ops)
allocate: Reuse the smallest freed number if any; otherwise use next_num and increment.

deallocate: Parse trailing digits as the number, remove from allocated, add back to free.

Example:

Query	Result
A apibox
apibox1
A apibox
apibox2
D apibox1
(frees 1)
A apibox
apibox1 (reused)
A sitebox
sitebox1
Output: ["apibox1", "apibox2", "apibox1", "sitebox1"]

3. Pairs with Difference k
Problem: Count pairs in array a whose difference equals k. Return the count mod 10⁹ + 7.

Solution
MOD = 10**9 + 7
def solution(k, a):
    count = {}
    for num in a:
        count[num] = count.get(num, 0) + 1
    result = 0
    for x in count:
        if x + k in count:
            result = (result + count[x] * count[x + k]) % MOD
    return result
How it was solved
Build a frequency map of each value.
For each value x, if x + k exists, add count[x] * count[x + k] pairs.
Because k ≥ 1, each pair is counted once (only when x is the smaller value).

Example: k = 3, a = [1, 6, 8, 2, 4, 9, 12]

x	x + k	Pairs
1
4
1 × 1 = 1
6
9
1 × 1 = 1
9
12
1 × 1 = 1
Output: 3

With duplicates, e.g. [3, 3, 6, 6] and k = 3 → 2 × 2 = 4 pairs.
